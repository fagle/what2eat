jquery.validator.js

简介

JS 表单验证插件，依赖于jQuery库。

目录

开发版：../assets/js/validator/jquery.validator.js

压缩版：../assets/js/validator/jquery.validator.min.js

接口

  1. $.setErrorStyle(style)：设置元素验证失败后的样式
  2. $.addRule(rule)：添加单个自定义验证规则
  3. $.addRules(rules)：添加多个自定义验证规则
  4. $(selector).verify([rules])：初始化元素验证或执行全局验证
  5. $(selector).disableVerify()：禁用元素验证
  6. $(selector).enableVerify()：启用元素验证

接口

$.setErrorStyle(style)

概述
设置元素验证失败后的样式
参数
style
元素验证失败后的 class 类名，默认为 has-error
示例
$.setErrorStyle('error');

$.addRule(rule)

概述
添加单个自定义验证规则
参数
rule
验证规则
示例
$.addRule({ name: 'required', title: '此项必填', rule: /.+/ });

$.addRules(rules)

概述
添加多个自定义验证规则
参数
rules
验证规则集合
示例
$.addRules([
	{ name: 'required', title: '此项必填', rule: /.+/ },
	{ name: 'number', title: '只允许输入数字', rule: /^(-?\d+)(\.\d+)?$/ }
]);

$(selector).verify([rules])

概述
初始化元素验证或执行全局验证
参数
selector
jQuery选择器，用于查找表单元素
rules
验证规则，可选
说明
不设置参数 rules 时表示执行全局验证，此时参数selector是指表单元素的父容器，如 $('body').verify() 即表示验证 body 容器内的所有待验证的元素，并返回一个布尔类型的变量，true 表示通过，false 则表示不通过；如果设置该参数，则表示给某个或某类元素加载验证，验证规则的格式分为三种，如下所述：
1、String 类型：'name' 或 'name1 name2 ...'
name 为规则名称，插件内部会根据该名称或名称集合匹配规则库中对应的验证规则来对表单元素加载单个或多个验证，如 $('#txtInteger').verify('required integer') 表示对 #txtInteger 文本框加载必填和整数验证；
2、Object类型：{ name: '规则名称', title: '失败提示', rule: 校验正则表达式或校验函数 }
使用这种参数形式可以自定义 title 失败提示和 rule 验证规则，如 $('#txtRequired').verify({ name: 'required', tips: '此项必填', rule: /.+/ })；
3、Array类型：['name', { name: '规则名称', title: '失败提示', rule: 校验正则表达式或校验函数 }]
为某个或某类表单元素加载多个验证，其中每项规则的设置同上 1、2 两点。
示例
$('#txtNumber').verify('required number');
$('#txtRequired').verify({ name: 'required', title: '此项不能为空' });
$('#txtNumber').verify(['required', { name: 'number', title: '此项只允许输入数字' }]);

$(selector).disableVerify()

概述
禁用元素验证
参数
selector
jQuery选择器，用于查找表单元素
示例
$('#txtNumber').disableVerify();

$(selector).enableVerify()

概述
启用元素验证
参数
selector
jQuery选择器，用于查找表单元素
示例
$('#txtNumber').enableVerify();

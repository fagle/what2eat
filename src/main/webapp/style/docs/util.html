util.js

简介

JS 工具包，提供常用的函数接口，不依赖于任何js库。

目录

开发版：../assets/js/util.js

压缩版：../assets/js/util.min.js

功能

  1. 命名空间类：注册命名空间
  2. 常规类：变量类型判断、创建GUID标识符、创建随机数、创建随机字符串等
  3. 输入检测类：判断用户的输入或选择是否正确
  4. 对象类：判断对象是否为空、对象深拷贝、判断对象是否包含指定的键、获取对象的属性长度等
  5. 数组类：判断指定元素是否存在于某个数组中、移除数组中的指定元素、数组去重等
  6. 数值类：将数值格式化为含有千分位的字符串、移除千分位等
  7. 字符串类：去除字符串左右空格、字符串截取后面加入...、将日期字符串转换为Date对象、格式化等
  8. 日期类：获取年、月、日、周以及当前时间等
  9. 运算类：求和、加、减、乘、除运算等
  10. 浏览器类：浏览器内核判断、名称获取、版本获取等
  11. 页面类：刷新当前页面、页面跳转等
  12. URL类：获取url参数集合、添加、修改或获取url指定参数
  13. DOM类：DOM载入就绪事件等
  14. 事件类：添加、删除事件处理函数等
  15. HTTP类：Ajax请求、Ajax请求队列等
  16. Cookie类：设置、获取和删除cookie
  17. Storage类：设置、获取和删除localStorage
  18. 移动类：根据屏幕大小动态设置rem相对单位

DOM载入就绪，类似jQuery的$(document).ready
示例
Util.dom.ready(function(){
	alert('ready');
});
事件类

Util.event.bind

概述
添加事件处理函数
示例
var btn = document.getElementById('submit');
var fn = function(){
	alert('您触发了按钮的点击事件');
};
Util.event.bind(btn, 'click', fn);

Util.event.unbind

概述
删除事件处理函数
示例
var btn = document.getElementById('submit');
var fn = function(){
	alert('您触发了按钮的点击事件');
};
Util.event.unbind(btn, 'click', fn);
HTTP类

Util.http.ajax

概述
发起Ajax请求，类似jQuery的$.ajax
示例
Util.http.ajax({
	method: 'GET',
	url: '/user.php?id=1001',
	data: null,
	success: function(xhr){
		...
	}
});

Util.cookie.remove

概述
设置cookie
示例
